# 🕉️ Kedarnath – Badrinath Pilgrimage 2026

An interactive, premium single-page travel guide, budget planner, and trek companion built in pure HTML, CSS, and JavaScript for the Himalayan pilgrimage of a lifetime.

## 🗺️ Route Overview
Mangalore → Delhi → Haridwar → Triyuginarayan → Gaurikund → **Kedarnath** → Vasuki Tal → Guptkashi → **Badrinath** → Charan Paduka → Mana Village → Joshimath → Rishikesh → Mangalore

🗓️ **Dates:** 13 June 2026 – 24 June 2026 (12 Days)

---

## 💎 Features

- **Interactive Day-wise Itinerary:** Expandable detailed timeline cards for all 12 days of the pilgrimage.
- **Dynamic Budget Planner:** Full expense breakdown, accommodation/food splits, and totals (Ultra-Budget vs. Comfortable).
- **Interactive Packing Checklist:** Category-wise items (Documents, Gear, Clothes, Meds) with a live progress bar. Auto-saves to `localStorage`.
- **Himalayan Trek Companion:** Detailed difficulty, distance, and safety tips for Kedarnath, Vasuki Tal, Charan Paduka, and Vasudhara Falls treks.
- **Live Gemini AI Chatbot (Yatra Companion):** 
  - Integrated directly into the page to answer pilgrimage questions.
  - Automatically switches to **Offline FAQ Mode** if there is no internet or if the API key fails.
  - Settings panel (⚙️) inside the chat window to configure/update your personal Gemini API key.
- **Trip Progress Tracker:** Mark days completed in the itinerary and watch your progress ring grow and unlock yatra milestone badges!
- **Print & PDF Support:** Clean custom styles optimized for printing or saving the itinerary directly as a PDF from your browser.
- **Premium Devotional Theme:** Elegant navy, saffron, and gold traditional color scheme with mountain silhouettes, animated stars, and mandala-inspired patterns.

## ⚙️ How to Run Locally

Since this is a **pure frontend application** (Single Page App), no complex server setup is required:

1. Clone this repository:
   ```bash
   git clone https://github.com/pratham-rai/Kedarnath-Badrinath.git
   ```
2. Open `index.html` directly in any web browser!

---

## 💬 Setting up the Gemini AI Chatbot

1. Open the website in your browser.
2. Click the chat icon (💬) in the bottom-left corner.
3. Click the gear icon (⚙️) in the chat header.
4. Paste your Gemini API key (you can generate a free key from [Google AI Studio](https://aistudio.google.com/)).
5. Click **Save Key**. The status will switch to **AI Online**!
